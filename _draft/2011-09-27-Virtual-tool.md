Понадобилось мне тут написать документацию по Active Directory, ибо хочется в отпуск, а без этой документации тут чёрт ногу сломит.

У меня уже в Virtual Box стоит Developer Preview Windows 2008, но на неё нивкакую не ставится Active Directory Topology Diagrammer. Пришлось разварачивать XP на виртуалке.

Зачем эта самя ADTD нужна?

Эта программа вытягивает из AD всю информацию по серверам, пользователям и структурным подразделениям. Формирует из этого некую диаграмму и визуализирует этот массив информации в Visio.

Таким образом можно быстро построить костяк для будущей документации по AD.
